# TechMind Academy - Tela de Login Responsiva 🔑

## 📖 A História do Projeto

A **TechMind Academy** é uma plataforma de cursos online voltada para tecnologia e inovação. Para proporcionar a melhor experiência desde o primeiro contato do estudante, foi solicitada a criação de uma interface de login que transmitisse modernidade, confiança e alta usabilidade.

Este projeto foi desenvolvido seguindo estritamente o briefing de identidade visual e requisitos estruturais da marca, superando os desafios clássicos do desenvolvimento front-end (como o alinhamento fluido do Flexbox, o posicionamento correto do rodapé e a adaptação inteligente de componentes entre telas grandes e dispositivos móveis).

## 🎨 Identidade Visual e Diretrizes

O projeto foi construído sob uma paleta de cores e tipografia planejadas para o nicho de tecnologia:
* **Cores:** Azul Vibrante (`#1E90FF`) para transmitir confiança; Laranja (`#FF6F00`) para botões de destaque; Fundo Cinza Claro (`#F5F5F5`) neutro e confortável para a visão.
* **Tipografia:** Fonte *Poppins* para títulos marcantes e *Roboto* para máxima legibilidade nos campos de texto.

## 📱 Estrutura e Engenharia de Responsividade

A interface foi projetada milimetricamente para se comportar de forma fluida e inteligente em três cenários:

* **Desktop (+1024px):** Layout dividido perfeitamente ao meio. O lado esquerdo exibe uma imagem inspiradora de tecnologia, enquanto o lado direito centraliza o formulário de login (com bordas arredondadas e sombra suave para dar sensação de profundidade).
* **Tablet (768px a 1024px):** Transição responsiva fluida, ajustando as margens para que o formulário continue centralizado e o rodapé fique no fluxo correto da página, sem quebrar os elementos.
* **Mobile (até 767px):** Experiência estilo "aplicativo nativo" (Full-Bleed). Para otimizar o espaço precioso do celular e garantir o conforto de digitação, a imagem lateral é ocultada, as bordas e sombras são removidas, e o formulário passa a ocupar 100% da tela com fundo branco unificado. Os botões sociais foram projetados para economizar espaço de forma discreta e elegante.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica da página de autenticação.
* **CSS3:** Estilização avançada com Flexbox, variáveis nativas (`:root`) e Media Queries cirúrgicas para quebras de tela.
* **Google Fonts:** Integração das fontes oficiais da marca (*Poppins* e *Roboto*).

## 🛠️ Como Executar Localmente

1. Faça o clone ou baixe este repositório.
2. Abra o arquivo `index.html` diretamente em qualquer navegador.
3. Utilize a ferramenta de inspeção do desenvolvedor (`F12`) para simular os dispositivos e testar o design responsivo.
